# Politica de Privacidad de NutrifynGo

Ultima actualizacion: 10 de abril de 2026

NutrifynGo es una aplicacion para registrar calorias, agua, pasos, rutinas y progreso diario.

## Datos que se procesan

- Datos del perfil ingresados por el usuario, como nombre, sexo, edad, peso, altura y objetivo.
- Datos de actividad fisica, como conteo de pasos, cuando el usuario concede el permiso correspondiente.
- Recordatorios y configuraciones locales de notificaciones.
- Imagen de perfil seleccionada por el usuario, si decide agregarla.

## Como se usan los datos

- Para mostrar el progreso diario dentro de la aplicacion.
- Para actualizar el widget de Android con informacion de calorias, agua y pasos.
- Para programar recordatorios y notificaciones configuradas por el usuario.

## Almacenamiento

- Los datos se almacenan localmente en el dispositivo del usuario.
- NutrifynGo no vende informacion personal a terceros.
- NutrifynGo no comparte datos personales con terceros con fines publicitarios.

## Permisos

- `ACTIVITY_RECOGNITION`: se usa para contar pasos y actividad fisica.
- `POST_NOTIFICATIONS`: se usa para mostrar recordatorios configurados por el usuario.
- `RECEIVE_BOOT_COMPLETED`: se usa para restaurar recordatorios despues de reiniciar el dispositivo.

## Control del usuario

El usuario puede:

- Editar sus datos personales en cualquier momento.
- Desactivar notificaciones desde la app o desde Android.
- Revocar permisos desde la configuracion del dispositivo.
- Desinstalar la aplicacion para eliminar los datos almacenados localmente.
