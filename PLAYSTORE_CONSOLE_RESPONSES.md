## 3. Permisos y justificacion

### ACTIVITY_RECOGNITION

Texto sugerido:

NutrifynGo usa este permiso para contar pasos y mostrar el progreso diario del usuario dentro de la app y en el widget.

### POST_NOTIFICATIONS

Texto sugerido:

NutrifynGo usa este permiso para enviar recordatorios configurados por el usuario, como hidratacion o seguimiento diario.

### RECEIVE_BOOT_COMPLETED

Texto sugerido:

NutrifynGo usa este permiso para restaurar recordatorios despues de reiniciar el dispositivo.
